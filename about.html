My name is Derek Duncan. I have been programming for the web since I was 11. With 5 years of experience, I have mastered HTML, CSS, Javascript, Ruby on Rails, and other web languages. I strive to use the latest techniques in website development.

I am currently residing in Tulsa, Oklahoma as a freelance website designer. I work to build websites that my clients love.

I started with my first project of building a family website. I spent many hours dragging and dropping widgets and element into a website builder program. When it came time to deploy the website, I found a host. Let's just say, deploying was much harder than I anticipated. After having the web host's support rep explain to me what HTML was, I got the HTML uploaded to the host. I became suddenly discouraged because what was on the website looked nothing like what I built in the website program.

From here, I ditched the drag-n-drop programs and started learning the code. It was a long process, with countless emails and phone calls to older friends in the business; thousands of Google queries; and hundreds of tutorials, I developed my first family website.

Building my first website from base code was an exhilarating process. From there I grew in my ability to program. I dove straight into CSS and Javascript to learn how to make a website 'pretty.' I began to dabble Serverside logic, and I soon learned Ruby on Rails. From there I have developed multiple custom Content Management Systems(CMS) for clients.

No doubt I am still learning to keep up with the latest standards, but those first years of website development prepped me for today and for the projects ahead.
